# Aether – Premium Music Streaming Landing Page

A sleek, high-fidelity UI/UX landing page design for a fictional premium music streaming application called **Aether**. This project focuses on modern dark-mode aesthetics, strict component architecture, and responsive layouts built directly within Figma.

Figma Link : 

https://www.figma.com/design/lLbulye37GSYgvch9nktyR/Untitled?node-id=0-1&p=f&t=tkswjtZST1qb0rdU-0

## 🌟 Design Aesthetic & Vibe
*   **Theme:** Ultra-Premium & Dark (Inspired by high-end audio platforms).
*   **Color Palette:** A sophisticated jet-black background paired with crisp white typography and a striking metallic gold accent to highlight key actions and premium tiers.
*   **Typography:** Clean, modern sans-serif geometric scale utilizing **Inter** to ensure maximum readability and an editorial feel.

---

## 🛠️ Design Implementations

### 1. Sections Implemented
*   **Hero Section:** Features a commanding value-driven headline (`LIFT OFF WITH SOUND. IMMERSE IN AETHER.`), an engaging subtitle, and a dual-button call to action (Primary Gold & Secondary Outlined) paired with abstract dark wave visuals.
*   **Featured Content:** "Featured Journeys" showcase displaying 4 curated playlists utilizing minimal, abstract art covers.
*   **Value Proposition:** Highlights 4 core features (*Ad-Free Listening*, *Hi-Res Audio*, *Exclusives & Live*, and *Offline Play*) mapped alongside representative iconography.
*   **Subscription Tiers:** Displays a clear multi-tier pricing layout matrix (*Basic*, *Premium*, and *Family*) featuring a distinct gold-flooded background accentuation on the recommended "Premium" tier.
*   **Social Proof:** "Hear the Echo" section hosting 3 individual horizontal testimonial cards complete with user profiles and quotes.
*   **Footer:** Organizes legal and functional site-map links (`About`, `Pricing`, `Support`, `Terms`) along with social media icons and a copyright notice.

### 2. Technical UI/UX Details
*   **Strict Auto Layout:** Applied universally across every frame, group, navigation bar, and component block (`Shift + A`) to ensure exact horizontal and vertical padding constraints.
*   **Component Library:** All atomic elements—including primary solid button variants, secondary outline buttons, navigation link bars, playlist items, and profile testimonial cards—were constructed manually from scratch.
*   **Responsive Variations:** Demonstrates responsive scalability with a dedicated Desktop (1440px) variant and a fully rearranged, vertically stacked Mobile Hero (390px) breakpoint frame.

---

## 🎨 System Specifications & Documentation

### Color Styles
*   **Background:** `#0A0A0A` (Deep Jet Black)
*   **Containers & Cards:** `#141414` (Subtle Charcoal)
*   **Primary Typography:** `#FFFFFF` (Pure White)
*   **Secondary Typography:** `#999999` (Muted Silver-Grey)
*   **Accent Color:** `#D4AF37` (Classic Metallic Gold)

### Typographic Scale
*   **H1 (Hero Headlines):** Inter / Bold / 64px
*   **H2 (Section Titles):** Inter / Semi-Bold / 36px (24px for Navigation)
*   **Body (Paragraph Text):** Inter / Regular / 18px
*   **Buttons & Interface Labels:** Inter / Medium / 16px
