## 📂 Repository Architecture & Contents

### 🌐 1. Application Development Tasks

* **📂 [HL](./HL/) — Domain-Specific App Dev Task**
  * *Project Name:* Hawkins Labyrinth
  * *Description:* A browser-based survival game set in the *Stranger Things* universe. Features a custom 5×5 canvas grid using pure CSS 3D transforms, real-time counter loops tracking enemy behavior, transparent visual elements, and an inventory power-up interface dock.
  * 🔗 **[Launch Hawkins Labyrinth Live Game](https://jolienes.github.io/Spider1/HL/)**

* **📂 [KOD](./KOD/) — Base App Dev Task**
  * *Project Name:* Keynesian Beauty Contest Game
  * *Description:* An algorithmic web application based on classic behavioral game theory models. Players input strategic numbers to guess against a dynamically shifting average controlled by an automated bot system. Outfitted with targeted input fields and explicit red border-glow buttons.
  * 🔗 **[Launch Keynesian Beauty Contest Live Game](https://jolienes.github.io/Spider1/KOD/)**

### 🛡️ 2. Cybersecurity Task

* **📂 [cyber](./cyber/) — Cybersecurity Base Task**
  * *Description:* Contains hands-on incident response logging analysis, system vulnerability audits, and technical patch workflow write-ups documenting modern defensive architecture.

### 🎨 3. UI/UX Design Task

* **📂 [uiux](./uiux/) — UI/UX Design Task**
